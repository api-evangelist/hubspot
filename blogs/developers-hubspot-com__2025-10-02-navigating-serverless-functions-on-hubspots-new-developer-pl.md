<p style="font-weight: bold; font-size: 14px;"><span style="font-style: italic;">Author's note (April 2026):&nbsp;</span></p> 
<blockquote> 
 <p style="font-size: 14px;"><span style="font-style: italic;">This post was originally published in October 2025 to help developers move off projects using built-in serverless functions to version <strong>2025.2</strong>, which did <strong>not</strong> support HubSpot-hosted serverless. As of <strong>platformVersion <code>2026.03</code></strong>, serverless functions are fully supported again for project-based apps, with app functions and public endpoint functions on the new Projects framework.</span></p> 
 <p style="font-size: 14px;"><span style="font-style: italic;"> For the current platform behavior, see the <strong><a href="https://developers.hubspot.com/docs/apps/developer-platform/add-features/serverless-functions/overview">serverless functions overview</a></strong> and <strong><a href="https://developers.hubspot.com/docs/apps/developer-platform/build-apps/migrate-an-app/overview">Determine your migration path to 2026.03</a></strong>.</span></p>  
</blockquote> 
<p>HubSpot just dropped what might be the most significant <a href="https://developers.hubspot.com/changelog/introducing-the-hubspot-developer-platform-2025">developer platform update</a> we’ve ever seen. From <a href="https://developers.hubspot.com/docs/apps/developer-platform/build-apps/create-an-app">developer projects</a> becoming the foundation for everything you build, to new features like<a href="https://developers.hubspot.com/docs/apps/developer-platform/add-features/custom-workflow-actions"> Custom Workflow Actions</a> and <a href="https://developers.hubspot.com/docs/apps/developer-platform/add-features/agent-tools/overview">Agent Tools</a>, to completely revamping our UI Extensions with dedicated<a href="https://developers.hubspot.com/docs/apps/developer-platform/add-features/ui-extensibility/create-an-app-home-page"> App Home pages</a>, the new platform represents a fundamental evolution in how we build on HubSpot.&nbsp;</p>
